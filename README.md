# Late Labels for Google Calendar

A Chrome Extension (Manifest V3) that injects humorous "arrival estimate" labels next to attendees in Google Calendar event details.

## Features
- **Humorous Estimates:** Adds lighthearted labels like "Usually 4m late" or "Typically on time".
- **Local Privacy:** All data is processed and stored locally within your browser.
- **Customizable:** Adjust humor levels or override estimates manually.

## Disclaimer
This extension is for entertainment purposes only. Arrival estimates are fictional or user-generated and are not based on actual tracking data.
