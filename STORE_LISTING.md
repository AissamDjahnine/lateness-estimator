# Chrome Web Store Listing

## Short description (up to 132 chars)
Playful lateness estimate chips for Google Calendar attendees. Edit labels; data stays local.

## Full description
Lateness Estimator adds compact, playful arrival-estimate chips next to attendee names in Google Calendar event details.

Features:
- Deterministic labels per attendee (e.g., "Usually 4m late", "Typically on time", "Will skip")
- Color-coded chips for quick scanning (green/orange/red)
- Click to edit labels; your edits persist locally
- Privacy-first: nothing leaves your browser

Notes:
- Works in the full event details dialog on calendar.google.com
- Labels are for humor and light productivity, not real tracking
